This website displays music artists and details about them.

To run this website:
1) Run: python webserver.py
2) Navigate to: http://localhost:5001/music/1/menu
3) Here you can perform CRUD operations with a successful Google login
4) Navigate to: http://localhost:5001/music/1/menu/JSON to see the JSON endpoint

To create the database:
1) Run: database_setup.py

To populate the database:
1) Run: populate_database.py
